Deliverable: Replace the contents of this element (.form) with a form that asks a user for a line for their favorite movie, book, or play.

Submit the sentence the user provided to the Yoda Speak API, and display the output to the user.

All HTML should be written in the index.html file, which is the file you're looking at now!

All CSS, if any, should be written in the styles.css file included in the project directory.

All JavaScript should be written in the script.js file included in the project directory.

Considerations:

  • What happens if the user doesn't specify anything?
  • What happens if they give you something that's not text?
  • What happens if there is an API error?
