Cornell DTI - Trends in Web Development Class Application

Deliverable

Replace the contents of this element (.form) with a form that asks a user for a line for their favorite movie, book, or play.

Submit the sentence the user provided to the Yoda Speak API, and display the output to the user.

How you choose to display this output to the user is entirely up to you; feel free to add as much or as little CSS as you want.

Submission

Submit this index.html file to the application when you are complete.

You can break up the JavaScript and CSS into different files. If that is the case, submit all your files as a ZIP file.

Considerations

  • What happens if the user doesn't specify anything?
  • What happens if they give you something that's not text?
  • What happens if there is an API error?
